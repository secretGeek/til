# What version of ubuntu am i running?

Use the command `lsb_release -a`

	$ lsb_release -a
	No LSB modules are available.
	Distributor ID: Ubuntu
	Description:    Ubuntu 16.04.6 LTS
	Release:        16.04
	Codename:       xenial
